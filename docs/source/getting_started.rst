Installation
------------

^^^^^^^^
Easy way
^^^^^^^^

Install ``runpandas`` with:

.. code-block:: bash

  pip install runpandas

RunPandas depends on the following packages:

- ``pandas``
- ``fitparse``
- ``stravalib``
- ``haversine``
- ``pydantic``
- ``pyaml``
